where it stands

The model today

After the burn, what remains is small and load-bearing. The hut is canon. When the hut and the running kernel disagree, the hut wins and the kernel follows. The garden is the downstream vault — aGarden — where actual runs live.

One kernel — generate → fitness → select → lens → judge → compile — runs in one direction at a time on a single dial: diverge for reach, converge for grounding. Fitness keeps novelty × grounding as two warring axes, never collapsed before selection, with a regret sibling that proves the dial mattered. A node is a projection of the RunTrace, never a second source of truth.

This log's siblings — the system-design document and the local field-lab app built alongside it — are downstream renderers of exactly this contract. They project the trace; the specimen is the trace.
